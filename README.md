La aplicación web optimizada para móviles que genera fondos de pantalla para teléfonos. Los usuarios describen el estilo que desean (por ejemplo, "lo-fi cyberpunk lluvioso") y la aplicación genera 4 variaciones en formato 9:16. Permite a los usuarios tocar una imagen para verla a pantalla completa con botones de "Descargar" y "Remezclar"; al remezclar, la imagen actual servirá de referencia para la siguiente tanda.
